# ITS
ICO smart contracts for TEU Pre-ITS
  
prequisite:  
* mist: https://github.com/ethereum/mist/releases  
* geth: https://github.com/ethereum/go-ethereum  
  
in your shell  
* run `# geth --fast --rinkeby`  
* launch `mist` app  
  
deploy the contracts  
* `TEUToken`   
* `teuTokenPreSale`
